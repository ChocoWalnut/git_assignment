git init – Initialize a new Git repository

git status – Check the status of files (tracked/untracked/modified)

git add <file> – Add file to staging area

git commit -m "message" – Save changes with a message

git log --oneline – View commit history in short form

git branch – List branches

git branch <name> – Create new branch

git checkout <name> – Switch to a branch

git merge <branch> – Merge branch into current branch

git log --oneline --graph --all – View full commit history graph

git revert HEAD – Undo last commit by creating a new commit

git remote add origin <url> – Connect local repo to GitHub

git push -u origin main – Push code to GitHub
