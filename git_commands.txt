git init
Initialized a new Git repository in the project folder to start version control.

git status
Checked the current state of the repository to see untracked and modified files.

git add index.html about.html
Added files to the staging area before committing.

git commit -m "Initial commit with index and about pages"
Saved the first version of the project with a meaningful message.

git branch feature
Created a new branch called 'feature' to work on changes separately.

git checkout feature
Switched to the feature branch to make updates.

git add about.html
Added updated about page content in feature branch.

git commit -m "Add feature branch content to about page"
Committed changes made in the feature branch.

git checkout main
Switched back to the main branch.

git merge feature
Merged feature branch changes into the main branch.

git log --oneline --graph --all
Viewed commit history in a simplified graph format.

git revert HEAD
Undid the last commit by creating a new commit that reverses it.

echo "*.log" > .gitignore
echo "temp/" >> .gitignore
Created a .gitignore file to ignore log files and temp folder.

git add .gitignore
git commit -m "Add gitignore for log files and temp folder"
Tracked and saved ignore rules.

git remote add origin https://github.com/ChocoWalnut/git_assignment.git
Connected local repository to GitHub remote repository.

git push -u origin main
Pushed all commits to GitHub and set upstream branch.
